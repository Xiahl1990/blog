2.环境变量: 以 npmconfig 为前缀的环境变量将会被认为是npm的配置属性。

3.用户配置文件:可以通过 npm config get userconfig 查看文件路径。

4.全局配置文件。可以通过 npm config get globalconfig 查看文件路径。

npm 相关问题

1.错误：“moudle不是内部或外部命令，也不是可运行的程序或批处理文件”
